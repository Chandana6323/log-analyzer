# Log Analyzer

A simple Python project to analyze log files and count errors, warnings, and info messages.

## Features
- Counts ERROR, WARNING, INFO logs
- Helps in debugging systems

## How to Run
1. Make sure Python is installed
2. Run:
   log_analyzer.py

## Sample Output
Errors: 2  
Warnings: 1  
Info: 2

## Tech Used
- Python
